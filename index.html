⚡ Funciones Lógicas: any() y all()
Preguntas de "¿hay algo?" y "¿hay todo?"
📋 any() vs all()
any() — "¿AL MENOS UNO?"
any([True, False])
# True ✅ (hay 1 True)

any([False, False])
# False ❌ (ninguno)

Devuelve True si AL MENOS UNO cumple

all() — "¿TODOS?"
all([True, True])
# True ✅ (todos True)

all([True, False])
# False ❌ (no todos)

Devuelve True si TODOS cumplen

Cómo se usan en FUNCIONES REALES:
password = "Abc12345"

# ¿Hay al menos UNA mayúscula?
any(c.isupper() for c in password)
# True ✅

# ¿Hay al menos UN número?
any(c.isdigit() for c in password)
# True ✅

# ¿Hay al menos UNA minúscula?
any(c.islower() for c in password)
# True ✅

# ¿TODOS son alfanuméricos (sin símbolos)?
all(c.isalnum() for c in password)
# True ✅
🎯 Ejemplo Completo: Ejercicio 8 (Validar Contraseña)
Cómo combinar TODAS estas funciones para resolver un problema real:
def es_contraseña_segura(password):
    """
    Usa: len(), any(), isupper(), isdigit()
    Comprueba que cumpla TODOS los requisitos
    """
    return (
        len(password) >= 8 and                      # ← len()
        any(c.isupper() for c in password) and      # ← any() + isupper()
        any(c.isdigit() for c in password)          # ← any() + isdigit()
    )

# Tests
print(es_contraseña_segura("abc123"))       # False (< 8 chars)
print(es_contraseña_segura("Abc12345"))     # True ✅ (cumple todo)
